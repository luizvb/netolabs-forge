# Forge brand system

**Positioning:** a controlled workspace to build, ground and evaluate dependable AI agents.

**Promise:** Build agents that hold up.

The mark represents two constraints compressing knowledge and instructions into a tested core. It avoids the usual robot, brain and sparkle shorthand and stays recognizable from a 16px favicon to a product header.

## Palette

- Forge Ink — `#20231F`
- Chalk — `#F2F3EE`
- Core Green — `#28624A`
- Grounded Sage — `#DCEBE2`
- Signal Lime — `#B8D94A`

## Typography

Manrope Variable is the primary typeface. The product uses sentence case; the wordmark and small operational labels use uppercase.
